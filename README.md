# Install BuLTM to your computer
1. Install support package. BuLTM is a Stan based package and to install BuLTM, you need to install related packages. It really depends on two packages, stan and rstantools. And you need to install these two packages first. 
For stan, you may refer to https://github.com/stan-dev/rstan/wiki/RStan-Getting-Started. 
For rstantools, you may install it by 
```
install.packages("rstantools")
```
2. Install by the source package. 
```
install.packages('BuLTM.tar.gz', repos = NULL, type = 'source')

```
